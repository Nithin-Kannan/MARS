Introduction

Welcome to our innovative project aimed at developing a cost-effective and low-maintenance articulated robotic system. Designed specifically for small-scale industries and startups, our system seeks to eliminate human error and enhance production rates. Our primary focus is on creating a prototype that incorporates a laser head, demonstrating the system's potential in precision tasks. This project aims to revolutionize small-scale production by providing an efficient, reliable, and adaptable robotic solution.

Key Features:

● Easily configurable with various end effectors, such as a laser head, drilling tool, pen holder, and more, to meet diverse user requirements.

● Capable of performing a wide range of tasks, making it suitable for confined spaces and particularly excelling in repetitive and monotonous work.

● Equipped with an intuitive and user-friendly interface, ensuring convenient operation for all users.

● High precision in tasks, especially with the laser head, ensuring meticulous and error-free results.

● Designed to grow with your business needs, accommodating additional functionalities and higher workloads as required.

● Simplified design for easy maintenance and minimal downtime, ensuring continuous productivity.

● Optimized to consume minimal power, reducing operational costs and environmental impact.

DESIGN OF MARS

Overview :

● The robot is designed in a way that most of the parts can be easily 3D printed. The joints are driven by NEMA 17 stepper motors which speed is reduced with the help of GT2 belts and some suitable 3D printed GT2 pulleys. The robot has 4 degrees of freedom, but in this case, we only used 3 DOF, two rotations for reaching any point in the work area and one linear motion for adjusting the height of the laser.

Marlin Firmware for Laser Engraving with SCARA Robot :

We used the Marlin firmware which is one of the most-widely used open-source firmware for 3D printers, which also has options for laser engraving and SCARA robot setups.



IDE For Uploading Marlin Firmware :

To install the Marlin firmware to the Arduino MEGA board we used PlatformIO IDE instead of Arduino IDE. As the Marlin firmware is big and complex, the Arduino IDE might sometimes give errors and not be able to compile it,PlarformIO is built on top of the Visual Studio Code editor, and it can be installed as extension to it, so we install Visual Studio Code, then search for the PlatformIO in the extension manager and install it from there.



Configuring The Marlin Firmware :

Marlin folder with the “SCARA” configuration files is opened in visual studio, in which the Configuration.h file parameters are edited according to our machine configuration.



Control Software - Rapetier-Host :

Rapetier host is used for controlling our robot, In printer configuration, Parameters corresponding to our Robot are added.



Generating G-code For Laser Engraving :

We are using Inkscape an open-source vector graphics software for generating G-code for MARS.
